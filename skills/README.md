# Official Agently Skills

This directory points to the published Agently skills catalog.

- GitHub: https://github.com/AgentEra/Agently-Skills
- Install: `npx skills add AgentEra/Agently-Skills`

Why install it:

- It gives coding agents concrete Agently implementation guidance instead of generic framework guesses
- It covers the main production surfaces, including model setup, output control, tools, MCP, FastAPIHelper, Session, and TriggerFlow
- It includes migration playbooks for LangChain and LangGraph teams moving into Agently
- It helps route requests to the right Agently abstraction faster, which reduces prompt overhead and rework

Use the official published skills repository instead of the old local docs or zip-based pack.
